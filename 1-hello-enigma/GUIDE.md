This project is a small Flask website that introduces who I am, my story, and the three core projects in this showcase. It uses a blue + cream colour palette and a personal profile picture (pfp.jpg) as the logo.

## Run the Project

1. Create and activate a virtual environment

macOS/Linux:
python3 -m venv venv
source venv/bin/activate

Windows:
python -m venv venv
venv\Scripts\activate

2. Install dependencies
pip install -r requirements.txt

3. Start the Flask website
python app.py

Then open in your browser:
http://127.0.0.1:5000/
